Luas Segitiga

Rumus Luas

L = 1/2 x a x t

Dimana:

L = Luas

a = Alas Segitiga

t = Tinggi Segitiga

Hitung Luas Segitiga

Keliling Segitiga

Rumus Keliling

K = a + b + c

Dimana:

a = Sisi 1

b = Sisi 2

c = Sisi 3

Hitung Keliling Segitiga

Luas Jajargenjang

Rumus Luas

L = a x t

Dimana:

L = Luas Jajargenjang

a = Alas Jajargenjang

t = Tinggi Jajargenjang

Hitung Luas Jajargenjang

Keliling Jajargenjang

Rumus Keliling

K = 2 x (AB + BC)

Dimana:

K = Keliling Jajargenjang

AB = Panjang Jajargenjang

BC = Lebar Jajargenjang
